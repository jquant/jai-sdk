JAI in 5 minutes
################

This section shows some end-to-end tutorials on how to use JAI. 
Here, you will have some examples of fitting different models, 
making inferences using JAI-SDK and directly using the JAI REST API.

.. note::

   and have a auth key before running the example codes on this section.

   If you haven't yet, please check :ref:`Getting Started <getting started>` section. |:smile:|
